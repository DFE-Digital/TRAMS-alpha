{{ serviceName }} Prototypes

This is where the Department for Education Regional Services Division (RSD) TRust and Academies Management System (TRAMS) prototypes for alpha phase live.

Version 3

Iterations on the prototype kit to include the 'find a trust journey':

  1. Header changes

    Included a search icon button in the header that when clicked displays a search bar or search page from non js users

  2. Changes to styles and text

    Changed banner colour and banner font colours so that error messages will have sufficient colour contrast

  3. Changes to 'find a trust'

    Changed search button icon to a green search button, included a search results page and added autocomplete to the search bar. This includes 2 possible search routes. If you start typing a trust name and select a trust in the auto-complete it will take you directly to the trust detail page. Alternatively you can just search for a term and click through to the search results page.

  4. Changed service name to 'Find information about academies and trusts'

Version 2

Iterations on the prototype kit to improve accessibility and usability across screen types:

  1. Develop style for statistics and figures

    In the first prototype some figures and statistics were laid out across two columns within grey cards. This layout changes significantly on smaller screens, and leads to unoptimal line lengths.

  2. Introduce dynamic data

    The use of dynamic data in templates will make future iterations on data tables easier for the team to develop, and will allow user testing with different scenarios.

  3. Update page headings and titles

    When a user is viewing information about a trust, information is split across several pages. Previously each page had the same title: the Trust's name. Now, the title and h1 both reflect the particular information a user is viewing about a trust, such as 'Governance', and a banner above the h1 displays the name of the Trust.

Version 1
